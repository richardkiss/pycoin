# ECDSA

The `pycoin.ecdsa` module provides ECDSA key and signature functionality.

## `__init__` Module

See `pycoin.ecdsa.__init__` for module documentation.

## Curve Module

See `pycoin.ecdsa.Curve` for module documentation.

## Generator Module

The `Generator` class represents an ECC generator point.

See `pycoin.ecdsa.Generator` for full class documentation.

## Point Module

The `Point` class represents a point on an ECC curve, supporting operations
`__add__`, `__mul__`, `__sub__`, and `__neg__`.

See `pycoin.ecdsa.Point` for full class documentation.
